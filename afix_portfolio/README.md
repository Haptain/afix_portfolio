## How to Use

### 1. Open in Browser
Simply double-click `index.html` or right-click and select "Open with" your preferred browser.

### 2. Add Your Photos
1. Copy your photos into the `images/` folder
2. Rename them to match the filenames above (or edit `index.html` to match your filenames)
3. Refresh the page - photos appear automatically!

### 3. Edit in VS Code
1. Open the `afix_portfolio` folder in VS Code
2. Edit `index.html` to customize text, add more units, or change styling
3. Use Live Server extension for real-time preview

### 4. Photo Naming Guide
Each photo placeholder shows the expected filename. If your photo has a different name, either:
- Rename your photo to match
- Or edit the `src="images/..."` path in `index.html`
